Процесс установки операционной системы семейства Linux (Ubuntu)

1. Создание мультизагрузочного usb-накопителя.

• Для создания мультизагрузочной флешки вам потребуется установленная на компьютере программа для создания, редактирования и конвертирования различных форматов CD и DVD-образов - UltraISO.

• Создание мультизагрузочной флешки производится в несколько этапов

- Подключение флешки к компьютеру

- Поиск образа диска с операционной системой

- Запуск программы UltraISO

- В меню программы выбрать «Файл» - «Открыть»

- Указать путь к файлу образа ОС

- В меню программы выбрать «Самозагрузка» - «Записать образ Жесткого диска»

- В поле Disk Drive(Диск) указать путь к флешке, на которую будет производиться запись.

- Далее поступит предупреждение о том, что флешка будет отформатирована, не забудьте сохранить данные с флешки.

- После завершения процесса создания вы можете пользоваться флешкой с целью установки ОС

• Примечание - Если мульти загрузочная флешка была выдана преподавателем этот пункт можно пропустить.

2. Выбор накопительного устройства для запуска операционной системы

• Включите компьютер, далее необходимо открыть BIOS(UEFI), для этого необходимо подать сигнал компьютеру клавишами DEL, F2 или F9 (Различается у разных производителей материнских плат).

• В BIOS необходимо открыть вкладку «Boot». Далее выбрать приоритет запуска накопительных устройств и выбрать мульти загрузочную флешку (Или дисковод, если установка производится с диска), после чего необходимо выйти из BIOS(UEFI), не забудьте сохранить изменения и перезагрузите компьютер.

3. Установка операционной системы

• После перезагрузки начнется установка операционной системы.

• При загрузке с носителя вам предложат выбрать язык интерфейса для работы с дистрибутивом Linux Ubuntu.

• После выбора языка необходимо выбрать режим «Установки» ОС нажатием на пункт «Установить Ubuntu»

• Следующим шагом будет выбор языка системы, раскладки клавиатуры.

• Далее необходимо выбрать приложения которые необходимо установить в процессе установки ОС, обычная установка или минимальная установка.

- Обычная установка – это установка системы со стандартным набором приложений. Рекомендована обычным пользователям ПК, так как в данном случае будут автоматически установлены все необходимые для работы программы;

- Минимальная установка – это установка системы с минимальным набором приложений. Данный режим Вы можете использовать, если Вам нужна чистая система только с основными утилитами, все приложения Вы будете устанавливать самостоятельно. Режим для тех, кто любит настраивать систему под себя, т.е. устанавливать только те приложения, которые ему нужны.

• После выбора режима установки, необходимо разметить жесткий диск, для этого необходимо

- Выбрать тип установки - «Другой вариант»

- Нажать на кнопку «Новая таблица разделов», предварительно выбрав устройство которое будет поделено на разделы

- Согласиться с созданием новой таблицы разделов

- Выбрать «свободное место» и нажать на кнопку «+»

- Для начала необходимо создать корневой раздел для самой системы. Для этого указываем: размер - минимум 10-15Gb, тип нового раздела - «Первичный», местоположение нового раздела - «Начало этого пространства», использовать как: - «журналируемая файловая система Ext4», Точка монтирования - указать «/».

- После необходимо создать домашний раздел для пользовательских данных. Для этого указываем: размер - оставшееся место, тип нового раздела - «Логический», местоположение нового раздела - «Начало этого пространства», использовать как: - «журналируемая файловая система Ext4», Точка монтирования - указать «/home».

- Разметка жесткого диска выполнена

• Для установки выбираем раздел с точкой подключения - «/»

• Далее программа установки спросит хотите ли вы записать внесенные изменения на диск, для этого нажмите на кнопку «Продолжить»

• В завершение установки необходимо выбрать часовой пояс, создать пользователя, задать имя компьютера и пароль для пользователя.

• После завершения установки необходимо перезагрузить компьютер.
